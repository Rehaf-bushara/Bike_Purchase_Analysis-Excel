# Bike_Purchase_Analysis-Excel
Excel dashboard analyzing bike purchase behavior based on customer demographics, income, commute distance, and car ownership.
## Project Overview

The main goal of this project is to explore patterns in bike purchases and visualize them using Excel PivotTables, charts, and slicers. The dashboard allows users to interactively filter by **Gender, Region, and Marital Status**.

**Key Features:**

- Interactive Excel dashboard with slicers
- PivotTables for data aggregation
- Charts answering specific questions
- Clean visual layout for easy interpretation
